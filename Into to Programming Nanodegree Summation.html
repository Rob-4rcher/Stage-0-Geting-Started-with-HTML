What is HTML
HTML is the standard mark-up language for creating web pages.

HTML consists of:

  • Text Content
  • Mark Up
  • Ref to Other Documents
  • Links

HTML uses mark ups to define contents these are called tags, these also help in defining white-space and how that space is viewed.
